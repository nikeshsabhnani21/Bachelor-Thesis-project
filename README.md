# Decision Making Under Uncertainty and Risk — Bachelor Thesis
Comparison of Bayesian Networks and Fuzzy Logic for emergency response decision-support, trained on a 1,048,575-entry dataset. Includes an Explainable AI (XAI) layer translating model outputs into clear actions ('Warning', 'Evacuation').

- Code.ipynb — full implementation and analysis
- Thesis Paper.pdf — full written thesis
- Bayesian Network Vizualization.png — model structure visualization

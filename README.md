# Decision Making Under Uncertainty and Risk — Bachelor Thesis

Comparison of Bayesian Networks and Fuzzy Logic for emergency response decision-support, trained on a 50,000-entry dataset. Includes an interpretable output layer translating probability outputs into clear action recommendations ('Warning', 'Evacuation').

- Code.ipynb — full implementation and analysis
- Thesis Paper.pdf — full written thesis
- Bayesian Network Vizualization.png — model structure visualization
